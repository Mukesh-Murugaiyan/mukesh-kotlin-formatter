## 📖 Alternative Usage (Command Palette)

If you prefer the Command Palette over shortcuts:
1. Open any `.kt` or `.kts` file in your editor.
2. Press <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd> (or <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd> on Mac).
3. Search for **Mukesh: Format Kotlin File 🚀**.
4. Hit **Enter** and enjoy perfectly formatted code!

---

## 🔧 Prerequisites

To guarantee top-tier formatting, this extension relies on the official `ktlint` engine.

- **ktlint** must be installed on your system.  
  *(Make sure it is available in your system's PATH)*

---

## 📦 Installation

Ready to elevate your Kotlin development?
1. Open Visual Studio Code.
2. Go to the Extensions view (<kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>X</kbd>).
3. Search for **Mukesh Kotlin Formatter**.
4. Click **Install**.
*(Alternatively, you can install it manually via the provided `.vsix` package.)*
